Application: https://smart-prep-vivek.vercel.app/

How to Run Web App with Code Editor Locally -

* Open Terminal and clone Git repository
* git clone "https://github.com/gaddamvivek/SmartPrep_Interview-Platform.git"
* cd SmartPrep_Interview-Platform
  
* First run the backend
* cd backend
* npm install
* node server.js

  
* Open New Terminal to Run Frontend
* cd frontend
* npm install
* npm start
